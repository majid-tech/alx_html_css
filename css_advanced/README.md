
# SmileSchool Webpage

A modern, responsive educational webpage for SmileSchool. This page showcases courses, tutorials, membership features, testimonials, and an FAQ section in a visually appealing and user-friendly layout.

## Features

- **Hero Section** with a bold call-to-action.  
- **Pros Section** highlighting instructors.  
- **Testimonials** with client feedback.  
- **Tutorials Section** showcasing popular tutorials with author info and ratings.  
- **Membership Section** presenting free membership features.  
- **FAQ Section** with common questions and answers.  
- **Footer** with social media links and logo.  
- Fully **mobile-responsive** with smooth scaling and layout adjustments.  

---

## Technologies Used

- HTML5  
- CSS3 (Flexbox, Grid, Media Queries)  
- Font Awesome icons for stars and social media.  
- Google Fonts: `Source Sans Pro` and `Poppins`.
